One-Time Setup Required

This app saves your data to guitar_practice_data.json in your Google Drive. You need a free Google OAuth Client ID.

  1. Go to console.cloud.google.com → create/select a project
  2. Enable the Google Drive API (APIs & Services → Library)
  3. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Application type: Web application
  5. Authorized JavaScript origins: add https://drive.google.com and null
  6. Copy the Client ID and paste it below

Your Client ID stays in your browser only.
